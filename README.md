# Homebrew tap for Mods

This tap installs the actively maintained `panjie/mods` fork.

```sh
brew install panjie/mods/mods
```

Or tap the repository first:

```sh
brew tap panjie/mods
brew install panjie/mods/mods
```

`homebrew/core` already has a `mods` formula for `charmbracelet/mods`, so use the
fully qualified `panjie/mods/mods` name to install this fork.
